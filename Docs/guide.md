# 欢迎使用 Simple Circuit Generator System

这是一个用来搭建和运行简单串联电路的小工具。你可以从左侧挑选元器件，把它们摆到网格画布中，再让系统自动连接和计算。

## 快速开始

1. 从左侧元器件库添加电源、灯泡、开关、电阻、电流表或电压表。
2. 拖动元器件，把主回路按蛇形顺序摆放整齐。
3. 如果添加了电压表，请先到“设置”中选择它要测量的元器件。
4. 点击右上角“运行”。
5. 在右侧“电路状态”面板查看电流、电压、电阻和运行状态。

## 怎样摆放元器件

当前版本喜欢整齐的蛇形队伍：

```text
第一行：从左向右
第二行：从右向左
第三行：再从左向右
```

系统会根据元器件中心的纵坐标判断它们是不是同一行。同一行的元器件尽量摆得水平一些，行与行之间也请留出足够距离。

画布左右和底部最好留一点空白，自动导线需要从这些位置转弯回家。

## 电路设置

打开“设置”窗口后，可以调整：

- **电源电压**：改变当前电源以及之后新建电源的默认电压。
- **电压表选择**：指定需要配置的电压表。
- **测量目标选择**：指定电压表测量哪个元器件。
- **电阻选择与阻值**：选择一个定值电阻并修改阻值。

设置电压表目标后，它会自动移动到目标上方。添加了电压表却还没有选择目标时，请先完成设置再运行，不然它会有一点不知所措。

## 运行结果怎么看

右侧状态面板会显示：

- 电路是否闭合运行。
- 当前总电压。
- 当前总电阻。
- 电流表示数。
- 电压表测量目标和示数。

如果有多个电压表，可以在下拉框中切换查看。没有电流表时，电路仍然能够计算，但面板会提示缺少电流表。

## 开关和灯泡

双击开关可以切换开合状态。

电路正常闭合并且存在有效电源、电阻时，灯泡会点亮；任一开关断开后，灯泡会熄灭。首次点击运行后，再切换开关就可以看到状态及时刷新。

## 清空画布

“清空”会删除当前元器件、导线、节点和运行结果，并把编号恢复到初始状态。准备搭建新电路时，按一下就能重新开始。

## 快捷键

| 操作 | 快捷键 |
| --- | --- |
| 运行 | `Ctrl+R` |
| 清空 | `Ctrl+L` |
| 设置 | `Ctrl+,` |
| 打开本说明 | `F1` |

## 当前版本的小约定

- 目前只支持自动生成简单串联回路。
- 暂时不能从端点手动拖线。
- 暂时没有撤回、重做和单独删除元器件。
- 导线会按照蛇形布局自动生成，因此摆放顺序越整齐，结果越漂亮。
- 并联电路和自由避障连线会在后续版本继续完善。

如果运行结果不符合预期，可以先检查开关是否闭合、电源是否存在、电阻是否大于零，以及电压表有没有选好目标。

慢慢摆，仔细看，让每一根导线都找到自己的位置。祝你搭电路顺利，灯泡一次点亮。
