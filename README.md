# Simple Circuit Generator System

## 项目简介

Simple Circuit Generator System 是一个基于 Qt 6 和 C++ 的简易电路图生成系统。项目当前围绕“可视化电路搭建”和“基础串联电路计算”展开：界面提供元器件库、绘图画布和运行入口，底层则用独立的电路模型保存元件、节点和连接关系，并通过求解器完成简单闭合串联回路的状态计算。

本项目适合用于入门级电路图绘制、电路元件建模和 Qt 图形视图框架练习。当前实现以基础功能为主，部分交互流程和扩展接口仍处于预留或逐步完善阶段。

## 当前功能概览

### 1. 图形化主界面

- 使用 `QMainWindow` 作为主窗口，左侧为元器件库，右侧为电路绘制画布。
- 画布基于 `QGraphicsView` 和自定义 `CircuitScene`，带有浅绿色网格背景，便于后续进行元件摆放和线路对齐。
- 顶部菜单栏右侧提供“运行”按钮，并带有快捷键和状态栏提示。
- 通过 `style.css` 统一管理界面样式，使元器件栏、按钮和画布保持一致的视觉风格。

### 2. 元器件库

项目中已定义并纳入界面的元器件类型包括：

- 电源
- 小灯泡
- 定值电阻
- 开关
- 电流表
- 电压表

这些元器件均有对应的资源图标或绘图类，用于在界面中展示，也有对应的电路模型类用于参与计算。

### 3. 图形元件绘制

`Graphics` 目录中封装了各类 `QGraphicsItem` 元件：

- `ComponentItem` 提供元件编号、可移动、可选中、左右端点位置和端点绘制等通用能力。
- 电源、灯泡、电阻、开关、电流表和电压表分别有独立的图形绘制类。
- 小灯泡支持亮灭状态显示。
- 开关图形支持开合状态展示，并预留了状态变化回调。
- 电流表和电压表图形类保留了读数更新能力。

这些图形类为后续实现拖拽摆放、连线编辑和运行后状态刷新提供了基础。

### 4. 电路数据模型

`Circuit_code` 目录中保存电路核心模型：

- `Component` 是所有电路元件的基类，保存元件编号、电阻值、左右连接节点等基础信息。
- `Node` 用于表示电路节点，并预留坐标信息以便和图形场景结合。
- `Connection` 用于把两个元件通过同一个节点连接起来。
- `Circuit` 统一保存元件和节点，并负责添加元件、添加节点、连接端点以及判断简单闭合回路。

当前模型主要面向简单串联电路，电压表被视为测量支路，不参与主回路导通判断和总电阻计算。

### 5. 简单电路求解

`CircuitSolver` 用于对当前电路进行基础计算。它的主要逻辑包括：

- 判断开关状态和电路是否形成闭合回路。
- 计算主回路总电阻。
- 汇总电源电压。
- 根据欧姆定律计算电流。
- 更新电流表读数。
- 根据电路是否导通更新灯泡亮灭状态。
- 根据电压表所测目标元件的电阻估算对应电压。

当电路未闭合、开关断开或总电阻无效时，求解器会将灯泡置为熄灭，并把电流表、电压表读数归零。

## 项目结构

```text
.
|-- Circuit_code/          # 电路元件、节点、连接、电路模型和求解逻辑
|-- Graphics/              # Qt 图形元件绘制类
|-- Cpp_Practice_Picture/  # 元器件图标和界面资源图片
|-- mainwindow.*           # 主窗口、元器件面板、运行入口和界面组织
|-- circuitscene.*         # 自定义电路画布与网格背景
|-- main.cpp               # Qt 应用入口和全局样式加载
|-- mainwindow.ui          # Qt Designer 生成的主窗口布局
|-- res.qrc                # Qt 资源文件
|-- style.css              # 界面样式
`-- CMakeLists.txt         # CMake 构建配置
```

## 构建环境

项目使用 CMake 管理构建，依赖 Qt 6：

- CMake 3.19 或更高版本
- Qt 6.5 或更高版本
- C++ 编译环境

可以通过 Qt Creator 打开项目，也可以使用 CMake 命令行进行配置和构建。

## 当前开发状态

当前版本已经完成了界面骨架、资源加载、基础图形元件、电路元件模型和简单串联电路求解逻辑。电路自动绘制、元件按钮到场景实例的完整交互、导线编辑、运行后图形状态同步等流程仍属于后续完善方向。
